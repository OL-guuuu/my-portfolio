## Notes for AI Agents

**You should be in Phase 1 mode:**

- **Do**: Implement approved scenes in static HTML/CSS, one at a time
- **Do**: Refine existing static foundations within scope
- **Do**: Preserve monochrome, premium, editorial aesthetic
- **Do**: Reference Lusion for quality/mood/pacing inspiration (never copy)
- **Don't**: Add interactions, transitions, or scroll choreography
- **Don't**: Introduce audio, loading experiences, or 3D/WebGL
- **Don't**: Work on more than one scene per task
- **Don't**: Expand scope beyond the approved scene

### Understanding Creative References

**Lusion (https://lusion.co/) is a reference for:**
- Quality bar and execution standards
- Atmospheric mood and depth
- Restraint and intentionality
- Cinematic pacing
- Premium editorial digital experience

**Lusion is NOT:**
- A template to copy
- A source for layouts or interactions
- Something to clone or replicate
- A reason to change our designer workspace world

**Our world remains:** A singular designer's office/workspace. Original. Unique.

### Before Taking Any Action

1. Confirm current phase is still Phase 1
2. Check if your planned action is allowed in Phase 1
3. Understand Lusion is for quality/mood reference only
4. If uncertain, ask before proceeding

### When Proposing Work

Make clear what phase your proposal belongs to. If it's not Phase 1 work, acknowledge that it will be deferred.
